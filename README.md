# 🌍 Outamaze – AI Tourism Recommendation System

An intelligent tourism recommendation platform that helps users discover travel destinations based on their interests and preferences. Built using Python and Flask, the application combines a responsive web interface with recommendation logic to simplify travel planning.

---

## 📌 Overview

Outamaze is a travel recommendation web application designed to provide personalized destination suggestions.

The platform allows users to explore tourist destinations through an easy-to-use interface while demonstrating full-stack web development using Python, Flask, HTML, CSS, and JavaScript.

## 🚀 Installation

Clone the repository

```bash
git clone <repository-url>

cd Outamaze-Tourism-Recommendation-System
```

Create virtual environment

```bash
python -m venv venv
```

Activate environment

Windows

```bash
venv\Scripts\activate
```

Linux/macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install flask
```

Run the application

```bash
python app.py
```
